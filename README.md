# 🗂️ 프롬프트 저장소 (Prompt Repository)

AI 프롬프트를 체계적으로 저장·관리·재사용할 수 있는 웹앱입니다.

## 주요 기능

- 📝 프롬프트 작성 (제목, 카테고리, 내용, 날짜 자동)
- 🔍 실시간 키워드 검색
- 🏷️ 카테고리 필터 (글쓰기 / 코딩 / 분석 / 번역 / 기타)
- 📋 원클릭 복사
- ✏️ 수정 / 🗑️ 삭제
- 🌙 다크 모드 (설정 자동 저장)
- 🔔 푸시 알림 (저장·복사 시 알림)
- 💾 localStorage 저장 (서버 없이 로컬 영구 보관)

## 기술 스택

- React 18 + Vite 5
- Tailwind CSS v4
- localStorage

## 실행 방법

```bash
npm install
npm run dev
```

브라우저에서 `http://localhost:5173` 접속
